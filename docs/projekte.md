# Projekte

**Version:** 1.0<br>
**Stand:** Mai 2026<br>
**Kurs:** Generative KI

Praxisorientierte Lernformate zum Aufbau von Agent-Systemen.

## Übersicht

### Workshop-Format
- **[RAG Workshop](projekte/rag-workshop.html)** – *Wie entsteht aus einzelnen Bausteinen ein zusammenhängender RAG-Workshop?*
  - Begleitete Schritt-für-Schritt-Übung
  - 9 Kapitel: vom einfachen Chatbot zur RAG-Anwendung mit Agent, Middleware und UI
  - RAG, SQL-RAG, Tool-Integration, Mini-Evaluation und Gradio-Oberfläche
  - Zeitaufwand: 6-8 Stunden
  - Module M04-M13

### End-to-End Projekt
- **[KI-Challenge](projekte/m23-ki-challenge.html)** – *Wie wird aus einem Kursprojekt ein belastbares End-to-End-System?*
  - Eigenständige Implementierung
  - 4 Projektoptionen: Support-System, Research-Team, Code-Review, Workflow-Automation
  - LangGraph Must-Haves: Supervisor-Pattern, HITL, Persistenz
  - Abgabe: Jupyter Notebook, SQLite-Datenbank und README
  - Zeitaufwand: 20-30 Stunden
  - Module M00-M20
